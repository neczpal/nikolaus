
# Nikolaus

A simple puzzle game based on the 
 [traveling salesman problem](https://en.wikipedia.org/wiki/Travelling_salesman_problem)
 starring Santa Claus who has to find the most optimal way
 of giving all children present and also return to Nordpole.

## Setup


* The project needs SDL2, SDL2_Image and SDL2_Gfx [libraries](http://www.libsdl.org).

  * Installing with Homebrew
      ```sh
      brew install sdl2
      brew install sdl2_image
      brew install sdl2_gfx
      ```
  * Or download them from the [official website](http://www.libsdl.org/download-2.0.php)

* Clone or download the repository.

* Build the project with [cmake](https://cmake.org)

* Run `Nikolaus` to play the game

* (Optional) Use `MapGen` with an input graph to get an output graph where the shortest route will be calculated
